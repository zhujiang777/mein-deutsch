# 德语学习经验、资料与 A1–B2 产品路线

> 综合小红书、YouTube 学习者经验和官方资料。个人进度并非效果承诺，课程或 App 推荐也可能含推广因素。

## 一、经验共识与分歧

1. **确定一条主线，再配少量工具。** 用一本教材或分级课程作骨架，以语法书、词典和真实材料补缺；不要频繁换教材。
2. **发音应尽早学，但不必一次完美。** 先学字母组合、长短音、重音和语调，再用模仿、录音或真人纠音修正；不可采用伤嗓子的极端训练。
3. **词汇要以可调用的词块记忆。** 名词连同冠词、复数和例句，动词连同搭配、支配格和句型学习；在语境中反复遇见，避免只背中文释义。
4. **语法不能跳过，也不应脱离语境。** 抓动词位置、格、时态等主干，在对话中观察后立即辨认、变换和造句；靠多轮重现掌握变格。
5. **听说应从 A1 开始。** 只读教材会留下明显短板。开口时先保证意思完整，再纠正反复出现、影响理解的错误。
6. **主动回忆比被动观看重要。** 精听可采用“听整句—暂停—复现—录音比较—看文本”；听写和影子跟读按目标选用。复习应覆盖旧词句、错题和输出，而非只打卡。

“语法—阅读优先”适合阅读目标，但需补听说；“输入—输出优先”适合交流目标，但需系统查漏。产品可允许不同侧重，同时保持能力底线。

## 二、A1–B2 分级路线

| 等级 | 学习目标与重点 | 推荐材料 | 产品能力 |
|---|---|---|---|
| A1 | 拼读、高频词块和生存交流 | Nicos Weg、VHS、Goethe 词表 | 中文提示、慢速对话、跟读、短语法、模板造句 |
| A2 | 扩展生活场景，叙述过去、计划和理由 | Nicos Weg、Momente/Menschen/Schritte | 连续听力、信息提取、对话分支、短文反馈 |
| B1 | 独立叙述、表达观点，接触分级真实材料 | Nicos Weg、Easy German、Logo! | 角色扮演、摘要、结构化写作、错误档案 |
| B2 | 理解新闻访谈，比较观点并进行论证 | 播客、Nachrichtenleicht、B2 样题 | 长输入、辩论、改写总结、正式写作、个性化纠错 |

## 三、贯穿四级的学习闭环

`场景输入 → 词块与发音 → 语法观察 → 检索练习 → 整句复述 → 情景输出 → 间隔复现`

- **A1：** 短对话后学习少量词块和一个语法点，经选择、重排进入模板造句、复述和角色扮演。
- **A2：** 对话变长，先抓关键信息；词块跨场景迁移，输出升级为短叙述和短消息。
- **B1：** 使用自然语速的分级内容，做摘要、说明理由、回应追问，并复习重复错误。
- **B2：** 使用新闻、访谈和多方观点，练习比较、论证、改写和正式表达；反馈后要求重写。

## 四、资料怎么搭配

| 资源 | 等级与作用 | 建议用法 |
|---|---|---|
| Nicos Weg | A1–B1 主线 | 借鉴故事场景、知识递进和综合练习，不只看视频 |
| VHS Lernportal | A1–B2 免费课程 | 补充系统练习和自测 |
| Momente / Menschen / Schritte plus Neu | A1–B1 教材参考 | 三选一比较场景、语法顺序和任务设计，不必同时学习 |
| Grammatik aktiv / 新求精语法 | A1–B1 语法 | 主线遇到问题后查阅并做针对性练习 |
| Goethe 词表与样题 | A1–B2 范围校验 | 用于查漏和阶段测试，不按词表顺序授课 |
| Easy German / Hueber Hören und Sprechen | A1–B2 听说 | 选择低于或接近当前水平的材料做复述、跟读和表达 |
| Logo! / Nachrichtenleicht | B1–B2 真实输入 | 先抓大意，再摘词块、复述和讨论，不逐词翻译 |
| 德语助手 / Anki / Quizlet | 全阶段工具 | 只保存真正遇到的词块，并保留语境和例句 |

## 五、对学习软件的长期建议

- 四个等级使用相同的单元循环，逐级增加输入长度、词汇密度、输出自由度和纠错深度，而不是做四套割裂课程。
- 词汇数据应包含冠词、复数、搭配、例句、语域、音频和来源场景，并支持从阅读、听力和错题回流到复习。
- 增加“听整句—口头重构—录音比较—显示文本”和情景角色扮演；语音识别只负责转写、对照和可解释反馈，不承诺可靠评判口音或流利度。
- 建立跨等级错误档案：只有重复出现或影响理解的发音、语法、搭配问题才提高优先级，避免每句话都被纠错打断。
- A1–A2 以强引导和即时反馈为主；B1–B2 增加摘要、主题讨论、观点论证、正式写作和修改重写。

## 六、可蒸馏对象与版权边界

优先研究 **Nicos Weg** 的场景顺序、故事推进和听说读写整合方式，以及 **Grammatik aktiv A1–B1** 的语法顺序和“解释—练习—输出”结构。B1–B2 再参考真实材料如何逐步增加长度、语速、抽象度和语域。只抽象知识范围、难度梯度和练习机制；不要复制教材原文、例句、音频、图片或成套习题，软件中的讲解、对话和练习应重新创作。
